# 手順

1. html ファイルと css ファイルを用意する
2. (1)を github.com のリポジトリにアップロードする
3. digitalocean.com から(2)のリポジトリをデプロイする
4. namecheap でドメインを買って DNS サーバーを設定する
5. digitalocean.com に(4)のドメインを設定する
