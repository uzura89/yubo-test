# 手順

1. html ファイルと css ファイルを用意する
2. (1)を GitHub のリポジトリにアップロードする
3. DigitalOcean で App を作成して(2)のリポジトリと連携させる
4. NameCheap でドメインを買って DigitalOcean と連携させる
5. DigitalOcean に(4)のドメインを追加する
